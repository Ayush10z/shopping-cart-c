# Shopping Cart Program in C 🛒

This is my **first C programming project**.

It is a simple console-based shopping cart application written in C that asks the user for:
- Item name
- Item price
- Item quantity

The program then calculates and displays the total cost.

## Features

- User input using `fgets()` and `scanf()`
- Basic arithmetic operations
- Formatted output using `printf()`
- Beginner-friendly C code

## Example

```
What would you like to buy?: Apple
What is the price of the item?: 25.50
What is the quantity of the item?: 4

Total = $102.00
```

## Technologies Used

- C Programming
- GCC (or any standard C compiler)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/shopping-cart-c.git
   ```

2. Navigate to the project folder:
   ```bash
   cd shopping-cart-c
   ```

3. Compile the program:
   ```bash
   gcc shopping_cart.c -o shopping_cart
   ```

4. Run the executable:
   ```bash
   ./shopping_cart
   ```
   On Windows:
   ```bash
   shopping_cart.exe
   ```
